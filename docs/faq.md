# Frequently asked questions

**Is this official?** No. **Will it edit my profile?** The public package APIs do not; never point custom adapters at a live profile. **Why does a note create multiple cards?** Templates (or cloze indexes) create study prompts. **Why not set due to a date directly?** Due units depend on queue and collection day. **Is CrowdAnki JSON exact?** No, it is a documented subset. **Can it read every APKG?** No; modern `collection.anki21b` is rejected explicitly.
